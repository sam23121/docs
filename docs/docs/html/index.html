Welcome to Lidar_Pipeline’s documentation!¶

Lidar_Pipeline is a Python Pipeline library for easily transforming and visulizaing lidar point clounds . It pulls data from the usgs 3DEP

<https://https://www.usgs.gov/>`_ and offers a simple and

intuitive API.

Note

This project is under active development.
